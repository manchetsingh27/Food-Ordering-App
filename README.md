## Overview

Foodly is a full-stack food ordering web application inspired by platforms like Zomato and Swiggy. The application allows users to browse food items, search restaurants, manage a shopping cart, place orders, track order status, and interact with an admin dashboard for food and order management.

The project was developed using Python Flask, SQLite, HTML, CSS, Bootstrap, and Jinja templates.



## Features

### User Features

* User Registration and Login Authentication
* Browse Food Menu
* Search Food and Restaurants
* Category-Based Filtering
* Add Items to Cart
* Remove Items from Cart
* Delivery Address Management
* Cash on Delivery Checkout
* Order Placement
* Order History Tracking
* Real-Time Order Status Updates

### Admin Features

* Secure Admin Login
* Admin Dashboard
* Add New Food Items
* Manage Menu
* View All Orders
* Update Order Status
* Food Inventory Overview

## Key Functionalities

### Food Discovery

Users can browse available food items, search by food name or restaurant, and filter by category.

### Shopping Cart

Users can add or remove items from their cart and review their total order amount before checkout.

### Order Management

Orders are stored in the database along with delivery addresses and current order status.

### Admin Control Panel

Administrators can add new menu items, monitor all orders, and update order statuses.
